# LP・チラシ制作ガイドライン

## 1. コピーライティング原則

### AIっぽさを避ける具体的な方法

**文体のバリエーション**
- 同じ長さの文を3つ以上並べない
- 体言止め、倒置、問いかけを混ぜる
- 接続詞を省略して、口語のリズムを作る

**具体性を優先**
- NG: 「主体性とコミュニケーション力を身につけた学生」
- OK: 「カフェ運営で買い出しから当日の進行まで自分たちで回してきた学生」

- NG: 「地域貢献に意欲的な学生」
- OK: 「近所の高齢者施設で、名前で呼ばれるくらい通い続けた学生」

**先生の声として自然に聞こえるか**
- 書いたコピーを声に出して読んでみて、不自然なら書き直す
- 「〜させていただきます」の多用を避ける
- 肩書きや実績の羅列より、エピソードで伝える

**NGワード・表現**
- 「画期的」「革新的」「唯一無二」
- 「Win-Win」「シナジー」
- 「〜力」の羅列（主体性・協調性・コミュニケーション力...）
- 箇条書きの各項目が同じ構文になること

### FAQ回答の書き方
- 定型的な回答を避け、実際に聞かれたときの返答のように書く
- 「はい、可能です。」で終わらせず、具体的な補足を入れる
- 先生が直接答えている感じを出す

## 2. デザイン原則

### やること
- 情報の優先度でサイズに差をつける（メインの活動 > サブの活動）
- 背景色の切り替えでセクションを分ける（#fff / #faf8f4 の交互）
- 余白で呼吸感を作る
- CTAボタンは目立つ色で1〜2箇所に絞る

### やらないこと
- 英語サブタイトル（"About Us", "Our Mission" 等）
- セクション間のdivider線や波形SVG
- グラデーション背景
- 装飾的な丸や図形（::before/::afterの飾り）
- 均等3カラムカードの繰り返し
- アイコンフォントの多用

### 色彩
- メインカラー: #2c6e4f（深緑、信頼感）
- アクセント: #c4593c（暖色、CTAに使用）
- サブカラー: #d4a24e（金茶、見出しアクセント）
- ベース: #faf8f4（温かみのあるオフホワイト）
- テキスト: #2d2d2d（真っ黒を避ける）

### フォント
- Hiragino Kaku Gothic ProN, Noto Sans JP, Meiryo, sans-serif
- 本文: 16px / line-height: 1.8
- 見出し: font-weight: 800〜900

## 3. 導線設計

### チラシ（flyer.html）の役割
- 対面で渡すので、自己紹介は先生が口頭で済ませる
- チラシは「後で見返したときに行動できる」ことが目的
- 載せる情報: キャッチコピー → 活動紹介 → メリット → CTA（QR+電話）
- 載せない情報: 趣味・特技・誕生日・研究テーマの詳細

### Web LP（lp/index.html）の役割
- チラシで興味を持った企業が「もっと詳しく知りたい」ときに見る
- 最終ゴール: Google Formでの問い合わせ
- 電話でも問い合わせ可能にする
- 教授の詳細紹介はページ下部（知りたい人だけ見る）

### Google Form
- 会社名・担当者・連絡先＋質問欄（作成済み）
- LP内にフォームを埋め込みor直リンク
