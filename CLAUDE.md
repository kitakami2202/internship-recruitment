# インターンシップ募集 LP・チラシ プロジェクト

## 概要
城西大学 柳澤智美教授のインターンシップ受け入れ企業募集のための制作物。
最終ゴールは「問い合わせ獲得」。

## 導線
対面（先生がチラシ手渡し）→ チラシのQRコード → Web LP → Google Form問い合わせ

## 制作物
- `flyer.html` — A4印刷用チラシ（募集特化、自己紹介は最小限）
- `lp/index.html` — Web LP（チラシで興味を持った企業向けの詳細ページ）

## コピーの原則
- AIっぽい均一な文体を避ける。文の長さやトーンにバリエーションを持たせる
- 抽象的な表現（「主体性」「コミュニケーション力」）→ 具体的なエピソードや場面で伝える
- 先生の言葉として自然に聞こえるトーンで書く
- 大仰な表現を避け、地に足のついた言葉を使う

## デザインの原則
- 英語サブタイトル、section divider、gradient、装飾的な::before/::after は使わない
- 完全対称レイアウトを避け、情報の優先度でサイズに差をつける
- 親しみやすいトーンを維持する

## 技術的な注意
- Google Form連携のJS部分はそのまま維持する
- QRコードは後日設定のプレースホルダー
- 電話番号: 049-271-8077
- 先生のイラスト/写真はプレースホルダーとして配置
